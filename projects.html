Ordixa

A multi-role workflow management system designed to manage assignment and order lifecycles with clear role separation.

Features include customer order creation, admin approval workflows, writer assignment, progress tracking, audit logs, secure session-based authentication, and data export.

Tech: Node.js, Express.js, HTML, CSS, JavaScript, session-based auth, JSON persistence.

PP Knows What You Think

A fun experimental web-based number guessing game that infers the user’s chosen number through a series of logical steps.

The core logic is implemented using bitwise operations, making it a playful exploration of logical thinking and JavaScript fundamentals.

Tech: HTML, CSS, JavaScript
